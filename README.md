## 功能
### 原理图
- ✅555定时计算器
- ✅LM317电压计算器
- ✅电池续航计算器
- ✅电容串/并联计算器
- ✅电阻串/并联计算器
- ✅分压电路计算器
- ✅LED串联电阻计算器
- ✅欧姆定律计算器

- ✅色环计算器
- ✅电流线宽计算器
